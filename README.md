# Redux Examples

Các ví dụ nhỏ sử dụng trong khóa học Redux

## Hướng dẫn

Cài đặt dependencies

```
yarn install
```

Khởi chạy server dev

```
yarn dev
```

Build production

```
yarn build
```

Test thử ở local

```
yarn serve -s
```

## Deploy lên Vercel

Đăng ký tài khoản [Vercel](https://vercel.com/dashboard)

Cài đặt [Vercel CLI](https://vercel.com/docs/cli)

```
npm i -g vercel
```

Đăng nhập

```
vercel login
```

Deploy

```
cd path/to/project

vercel
```
